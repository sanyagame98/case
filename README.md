# CaseVerse Mono Stable

Локальный проект с кейсами и виртуальной валютой.

## Что исправлено
- переработана верстка, чтобы текст не вылезал за границы
- стабилизированы вкладки и кнопки открытия кейсов
- улучшена лента открытия кейса
- модалки закрываются корректно
- карточки предметов полностью залитые, без иконок

## Запуск
Открой `index.html` в браузере.

Для более стабильной работы можно использовать локальный сервер:

```bash
python -m http.server 8000
```

Потом открой `http://localhost:8000`
